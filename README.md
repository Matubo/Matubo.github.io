# pokemonrequest

Страница обращается к открытому RESTful API - [pokeapi.co](https://pokeapi.co/).
Она позволяет найти и вывести информацию о покемонах по кретериям указанным ниже.

### Критерии поиска:
- id в базе данных
- диапазон id в базе данных
- имя
